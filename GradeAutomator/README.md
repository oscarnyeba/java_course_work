# GradeAutomator

Simple Java console app that calculates and displays grade distribution for students.

## What It Does

- Takes marks for 5 students (0-100)
- Assigns grades based on mark boundaries
- Shows how many students got each grade

## Grade Scale

| Grade | Mark Range |
|-------|-----------|
| 1     | 80-100    |
| 2     | 75-79     |
| 3     | 66-74     |
| 4     | 60-65     |
| 5     | 50-59     |
| 6     | 45-49     |
| 7     | 35-44     |
| 8     | 30-34     |
| 9     | 0-29      |

## How to Run

```bash
# Compile
javac ./src/gradeautomator/GradeAutomator.java

# Run
java gradeautomator.GradeAutomator
```

## Example

```
=== Welcome to GradeAutomator ===
Enter marks for student 1: 85
Enter marks for student 2: 72
Enter marks for student 3: 65
Enter marks for student 4: 58
Enter marks for student 5: 42

===Grades and number of students who attained them===
Grade===Students
1===1
2===1
3===0
4===1
5===1
6===1
```
