# OOP Course Work - Java Applications

Collection of console and GUI applications developed for Object Oriented Programming course at Victoria University.

## Projects

### 1. GradeAutomator
Simple console app that calculates grade distribution for 5 students.
- Assigns grades (1-9) based on mark boundaries
- Displays grade distribution summary

### 2. Student Registration App
JavaFX desktop application for student registration with data persistence.
- Validates personal details (email, password, age)
- Auto-generates unique student IDs (YYYY-NNNNN format)
- Saves registrations to CSV file

### 3. SK Motors Auction System
Vehicle auction tracking system that calculates profit/loss.
- Registers vehicles and collects bids from 3 bidders
- Identifies highest bidder
- Computes profit/loss on sale
